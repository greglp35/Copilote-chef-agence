# AJI Studio - Dossier terrain

Objectif : centraliser le travail AJI Studio pour preparer les imports, catalogues, publicites, controles et exports.

## Parcours terrain

1. Ouvrir AJI Studio.
2. Importer le JSON de travail depuis le dossier imports.
3. Corriger les produits bloques dans le plan 48h.
4. Completer les visuels produits et packs.
5. Valider prix, stock, unite, conditionnement et mentions.
6. Passer uniquement les lignes validees en usage_public oui.
7. Generer catalogues, affiches, posts ou fiches pack.
8. Archiver les exports dans le dossier exports.

## Regles de travail

- Pas de comparaison concurrentielle sans preuve datee.
- Pas de support public si prix, stock ou unite sont incertains.
- Pas de donnee interne dans un support public.
- Les fichiers de travail doivent rester versionnes.
