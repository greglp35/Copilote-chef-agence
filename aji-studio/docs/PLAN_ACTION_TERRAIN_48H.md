# Plan action terrain 48h - AJI Studio

But : rendre le dossier exploitable rapidement en agence.

## J0 - Mise en place

- Ouvrir AJI Studio.
- Importer le JSON de travail.
- Verifier que les produits apparaissent.
- Verifier que le bouton Plan commercial fonctionne.
- Sauvegarder une version JSON de depart.

## J1 - Fiabilisation produit

- Traiter les produits avec prix manquant.
- Completer les unites de vente.
- Completer les conditionnements.
- Identifier les produits sans photo.
- Classer les produits par pack suggere.

## J2 - Preparation publicite

- Choisir 5 a 8 produits a pousser.
- Choisir 3 packs prioritaires.
- Valider les accroches A3 et comptoir.
- Ajouter les mentions legales.
- Exporter un PDF test.

## Regle de sortie publique

Un produit ne passe en public que si les champs suivants sont valides : prix, unite, stock ou delai, visuel, mention, usage_public.

## Resultat attendu

A la fin des 48h, l'agence doit avoir :

- une base importee ;
- une liste de corrections restantes ;
- des produits prioritaires ;
- des packs validables ;
- un premier support PDF test.
