# Exports AJI Studio

Ce dossier sert a archiver les sorties generees depuis AJI Studio.

## Types d'exports attendus

- PDF test interne
- PDF public valide
- JSON sauvegarde projet
- CSV produits corriges
- catalogues numerotes
- affiches A3
- posts reseaux sociaux

## Regle de nommage

YYYY-MM-DD_type_support_version_statut

Exemples :

- 2026-05-15_catalogue_printemps_v1_interne.pdf
- 2026-05-15_affiche_terrasse_v1_public.pdf
- 2026-05-15_sauvegarde_aji_studio_v1.json

## Regle de validation

Un fichier public ne doit etre ajoute ici que si la checklist publication est validee.
