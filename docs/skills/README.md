# Skills — Copilote Chef d’Agence

Ce dossier regroupe les compétences utiles au développement des applications HTML métier du projet Copilote Chef d’Agence.

Objectif : éviter de partir dans des compétences trop larges ou trop techniques, et conserver uniquement ce qui sert le cas d’usage réel : agence de négoce matériaux, stock, exploitation, TFI, imports/exports, sécurité, validation humaine et outils terrain.

---

## Fichier principal

```text
MATRICE_COMPETENCES_COPILOTE_AGENCE.md
```

Ce fichier sert de grille de décision pour savoir quelles compétences développer, rechercher, intégrer ou refuser.

---

## Compétences prioritaires

1. HTML/CSS/JS autonome.
2. JSON Schema et contrats de données.
3. Stock mini/maxi négoce matériaux.
4. Mapping CSV / TFI.
5. Sécurité HTML et anti-secrets.
6. UX terrain mobile.
7. Recette GitHub Actions.
8. Documentation exploitable par IA.

---

## Règle simple

Une compétence est utile uniquement si elle améliore au moins un de ces points :

- usage terrain ;
- sécurité ;
- fiabilité des données ;
- traçabilité ;
- simplicité ;
- validation humaine ;
- capacité à être reprise par Codex ou ChatGPT.

Sinon, elle est à reporter.
